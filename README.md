## 📁 Project Overview

This project contains several data analysis tasks using dummy datasets.

### 🔹 Task 1
- Average calculation by city using `AVERAGEIF`
- Total transaction based on payment method using `SUMIF`
- Unique data calculation using array formula
- Sales filtering using `SUMIFS` with date criteria
- Identifying top product based on quantity

### 🔹 Task 2
- Sales calculation for specific categories
- Average analysis based on gender & category
- Monthly sales analysis using Pivot Table
- Identifying highest transaction category

### 🔹 Task 3
- Multiple analytical queries and problem solving using Excel formulas

### 🔹 Task 4
- Total sales based on payment method
- Best-selling product analysis
- Monthly transaction summary
